## Getting Started

### 1. 프로젝트 클론

```bash
git clone https://github.com/Younggun-Kim/Calculator.git
```

### 2. 라이브러리 설치
```bash
flutter pub get
```

### 3. 프로젝트 내에서 `calculator_main.dart` 실행

```bash
dart run lib/calculator/calculator_main.dart
```

### 4. 테스트 실행

```bash
flutter test
```
